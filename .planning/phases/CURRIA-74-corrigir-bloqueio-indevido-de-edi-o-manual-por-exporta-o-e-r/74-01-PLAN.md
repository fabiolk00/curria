# 74-01 Plan

## Goal

Remove the save/download dead zone caused by export lock conflicts during manual resume editing.

## Steps

1. Narrow session-generate conflict detection to the current session or target scope only.
2. Preserve the current valid artifact when a same-scope export is genuinely active during manual save.
3. Update the editor modal so save success is not reported as failure when PDF refresh is merely deferred by an active export.
4. Add structured observability for manual-save lock detection and unavailable-download states.
5. Add regression tests for the real broken flow.
