# 58-01 Plan

## Objective

Harden the post-ATS handoff into `generate_file` so the authoritative source is explicit, coherence is checked before export, typed failures survive to the route layer, and the recurring failure class is regression-tested.

## Tasks

1. Add an explicit `generate_file` intake resolver that derives the authoritative export source from session-backed state and rejects mismatched request snapshots.
2. Add a smart-generation post-persistence handoff preflight before dispatching `generate_file`.
3. Preserve typed tool failure codes through smart-generation and legacy ATS route normalization instead of flattening dispatch failures to generic 500s.
4. Add regression tests for payload/source mismatch, post-persistence handoff failure, typed route failure mapping, and the preview-lock transverse flow.
5. Add structured preflight logs and seam metrics so future intake failures are self-explanatory.
6. Run `npm run typecheck`, `npm run audit:route-architecture`, targeted seam tests, and `npm run test:architecture-proof-pack`.
