# Quick Task: Remover scroll das telas de auth

## Objetivo

Garantir que `/entrar` e `/criar-conta` não tenham barra de rolagem vertical ou horizontal em viewports responsivos comuns.

## Plano

- Travar o shell em `100dvh` com `overflow-hidden`.
- Remover altura mínima rígida do card principal.
- Compactar espaçamentos, logo e campos de formulário com medidas responsivas.
- Ocultar copy secundária/footer em alturas baixas para preservar o formulário sem scroll.
- Validar em navegador com múltiplos viewports.

## Validação

- `npm run typecheck`
- `npm run lint`
- `npm run build`
- `npm test`
- Medição Playwright de `scrollHeight`, `clientHeight`, `scrollWidth` e `clientWidth` em `/entrar` e `/criar-conta`.
