# Quick Task: Aplicar PNG separado como favicon e ícone do sidebar

## Objetivo

Usar o PNG separado enviado na raiz como favicon do site e substituir o ícone antigo de robô no sidebar.

## Plano

- Localizar o PNG enviado na raiz do projeto.
- Copiar o PNG para `src/app/icon.png` para favicon App Router.
- Copiar o PNG para `public/trampofy-icon.png` para uso em componentes.
- Remover o favicon SVG antigo.
- Atualizar `SiteFaviconIcon` para renderizar a nova imagem.
- Atualizar metadata para apontar para o favicon PNG.
- Validar typecheck, lint, build e assets no localhost.

## Validação

- `npm run typecheck`
- `npm run lint`
- `npm run build`
- Conferir `/icon.png` e `/trampofy-icon.png` no localhost.
