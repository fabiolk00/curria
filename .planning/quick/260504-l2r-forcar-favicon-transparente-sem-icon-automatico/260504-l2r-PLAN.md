# Quick Task: Forçar favicon transparente

## Objetivo

Garantir que o favicon servido pelo site use o PNG transparente, evitando a rota automática `/icon.png` que estava servindo uma versão antiga opaca no dev server.

## Plano

- Verificar alpha do PNG local e do PNG servido.
- Remover `src/app/icon.png` para desativar o ícone automático do App Router.
- Manter metadata apontando para `/trampofy-icon.png`.
- Limpar cache `.next` do dev server e reiniciar localhost.
- Confirmar links de favicon no HTML e alpha do asset servido.

## Validação

- `npm run typecheck`
- `npm run lint`
- `npm run build`
- Conferir `<link rel="icon">` em `/termos`.
- Conferir alpha do PNG servido por `/trampofy-icon.png`.
