# Quick Task: Usar texto Trampofy em seção ATS

## Objetivo

Substituir a logo inline no título "Como a Trampofy te ajuda a vencer" por texto simples.

## Plano

- Editar `src/components/landing/o-que-e-ats-page.tsx`.
- Remover import de `BrandWordmark`.
- Trocar `<BrandWordmark />` por `Trampofy`.
- Validar typecheck, lint, build e localhost.

## Validação

- `npm run typecheck`
- `npm run lint`
- `npm run build`
- Conferir `/o-que-e-ats` no localhost.
