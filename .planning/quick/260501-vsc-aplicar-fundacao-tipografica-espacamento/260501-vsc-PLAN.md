# Quick Task 260501-vsc: Global design foundation

## Goal

Aplicar a direcao visual fornecida como fundacao do projeto inteiro, usando tokens globais, Tailwind theme e componentes base compartilhados em vez de ajustes isolados por tela.

## Tasks

1. Definir tokens globais de cor, tipografia, espacamento, radius e shadow em `globals.css` e expor aliases no Tailwind.
2. Atualizar primitives compartilhados (`Button`, `Card`, `Badge`, inputs, dialogs, tabs, tooltip etc.) para consumir a fundacao.
3. Rodar validacao focada de typecheck, testes de UI/pricing/resume afetados e smoke visual basico.
