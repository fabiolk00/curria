# Quick Task: Adicionar sidebar de seções na privacidade

## Objetivo

Fazer a página `/privacidade` seguir o padrão visual/interativo de `/termos`, com navbar público e menu lateral clicável de seções.

## Plano

- Criar componente client para a página de privacidade.
- Mover seções da política para uma lista estruturada.
- Reaproveitar o padrão de sidebar sticky com item ativo via `IntersectionObserver`.
- Manter a rota server apenas para metadata.
- Validar ausência de mojibake, typecheck, lint, build e localhost.

## Validação

- Busca por caracteres corrompidos.
- `npm run typecheck`
- `npm run lint`
- `npm run build`
- Conferir `/privacidade` no localhost.
