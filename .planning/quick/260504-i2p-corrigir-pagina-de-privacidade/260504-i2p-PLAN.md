# Quick Task: Corrigir página de privacidade

## Objetivo

Corrigir a página `/privacidade`, removendo mojibake, preenchendo a marca Trampofy no texto legal e substituindo o uso problemático de logo inline por texto normal.

## Plano

- Abrir a rota pública de privacidade.
- Remover `BrandText` do corpo jurídico.
- Reescrever a copy em pt-BR sem caracteres corrompidos.
- Adicionar o header público para usar a logo atual.
- Validar typecheck, lint, build e localhost.

## Validação

- `rg` para caracteres corrompidos na página.
- `npm run typecheck`
- `npm run lint`
- `npm run build`
- Conferir `/privacidade` no localhost.
