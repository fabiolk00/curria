# Quick Task: Ajustar breakpoint da navbar pública

## Objetivo

Evitar quebra da navbar pública em larguras intermediárias, exibindo hamburger abaixo de `lg`.

## Plano

- Reescrever `src/components/landing/header.tsx` em UTF-8 limpo.
- Alterar navegação desktop de `md:flex` para `lg:flex`.
- Alterar hamburger de `md:hidden` para `lg:hidden`.
- Exibir botões completos de auth só em `lg+`.
- Validar typecheck, lint, build e viewport de 811px.

## Validação

- Busca por mojibake no header.
- `npm run typecheck`
- `npm run lint`
- `npm run build`
- Playwright em 811x918: nav desktop oculto, menu visível, sem overflow horizontal.
