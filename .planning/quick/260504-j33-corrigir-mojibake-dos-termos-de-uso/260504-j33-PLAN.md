# Quick Task: Corrigir mojibake dos Termos de Uso

## Objetivo

Corrigir todos os caracteres quebrados na tela `/termos`, mantendo a estrutura de sidebar, header público e conteúdo jurídico existente.

## Plano

- Reescrever `src/components/terms/terms-page.tsx` em UTF-8 limpo.
- Manter a lista de seções, sidebar clicável e destaque ativo.
- Corrigir ids com acentos para ids ASCII estáveis.
- Validar ausência de mojibake, typecheck, lint, build e localhost.

## Validação

- Busca por caracteres corrompidos no fluxo de Termos.
- `npm run typecheck`
- `npm run lint`
- `npm run build`
- Conferir `/termos` no localhost.
