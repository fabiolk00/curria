# Quick Task: Substituir telas entrar e criar-conta pelo login do zip

## Objetivo

Aplicar o layout de autenticação enviado no zip às rotas públicas `/entrar` e `/criar-conta`, preservar a autenticação real existente e trocar a marca visual pelo PNG novo da Trampofy.

## Escopo

- Extrair a referência visual do zip enviado na raiz do projeto.
- Copiar o PNG oficial da Trampofy para `public/`.
- Atualizar o componente global de logo para usar o PNG com iconografia e wordmark.
- Substituir a casca visual compartilhada de login/cadastro por um layout inspirado no zip.
- Traduzir e adaptar a copy pública para pt-BR.
- Preservar formulários, redirects e integração de autenticação existentes.

## Validação

- `npm run typecheck`
- `npm run lint`
- `npm test`
- `npm run build`
- Conferir `/entrar` e `/criar-conta` no localhost.
