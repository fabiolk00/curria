# Quick Task: Substituir logo PNG pelo SVG da root

## Objetivo

Usar o SVG oficial presente na raiz do projeto como fonte da marca Trampofy no componente global de logo.

## Plano

- Localizar o SVG oficial na raiz do projeto.
- Disponibilizar o SVG em `public/` para uso pelo navegador.
- Atualizar `BrandWordmark` para apontar para o SVG.
- Validar build e localhost.
