# Plan 13-02

## Goal

Verify and harden server-only secret boundaries across the app.

## Tasks

1. Audit imports and runtime boundaries for service-role keys, Clerk secrets, OpenAI keys, Asaas tokens, and similar server-only credentials.
2. Add or strengthen server-only assertions where the current boundary is implicit.
3. Remove or refactor any ambiguous shared utility path that can cross into client or edge bundles.
4. Document the intended secret-boundary model for future contributors.

## Verification

- Focused audit findings closed in code or documentation
- Tests or build checks for server-only module boundaries where practical
- No known ambiguous secret path left untracked
