# Plan 12-01

## Goal

Add bounded OpenAI timeout and circuit-breaker behavior to the critical resume-generation paths.

## Tasks

1. Identify the OpenAI call sites that can cascade across ATS enhancement, job targeting, and file generation.
2. Introduce a shared provider-protection utility with:
   - timeout budget per request stage
   - bounded retries with backoff
   - circuit-breaker state with open, half-open, and close transitions
3. Wire the utility into the highest-risk generation paths first without changing user-visible happy-path behavior.
4. Emit structured logs for breaker opens, timeouts, and degraded fallbacks.

## Verification

- Unit tests for breaker state transitions and retry ceilings
- Route or pipeline tests proving provider slowness does not hang the full flow
- Structured-log assertions for timeout and breaker events
